Conceptos

Desarrollador Web

Es la persona que se encarga de realizar una página web, tanto en funcionalidad como en aspecto.

Diseñador web

Se encarga de dar forma a la página web y diagramarla, luego le transmitirá ese diagrama al desarrollador para que lo lleve a cabo.

Programa de computadora

Son instrucciones que se le dan a una computadora para que las ejecute.

Algoritmo

Son instrucciones que se siguen para resolver un problema.

Lenguaje de programación

Es un lenguaje creado con la capacidad de describir acciones y que una computadora lo pueda interpretar y ejecutar.

Lenguaje de alto nivel

Este tipo de lenguaje se acerca al lenguaje humano, la computadora debe traducirlo a un lenguaje de bajo nivel. Algunos lenguajes de alto nivel son JavaScript, PHP, SQL, etc.

Lenguaje de bajo nivel

Este tipo de lenguaje se acerca a la computadora, es más difícil para aprenderlo y corregir errores.

HTML

Son una serie de etiquetas utilizadas para crear una página web.

CSS

Son una serie de etiquetas utilizadas para darle forma, color y efectos a una página web.

JavaScript

Es un lenguaje de programación que se usa para darle funcionalidad a la página web.
